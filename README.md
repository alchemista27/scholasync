# ScholaSync

Project ini adalah platform komprehensif yang dirancang untuk menyederhanakan dan memodernisasi administrasi sekolah melalui dua fitur utama:

1.  **Absensi Berbasis GPS**: Sistem absensi seluler yang memungkinkan pelacakan kehadiran siswa dan staf secara akurat menggunakan lokasi geografis.
2.  **Penerimaan Peserta Didik Baru (PPDB) Online**: Sebuah sistem untuk mengelola seluruh proses pendaftaran siswa baru, mulai dari pendaftaran, seleksi, hingga pengumuman hasil secara online.

## Teknologi

-   **Backend**: Supabase (PostgreSQL, Auth, Storage)
-   **Frontend (Admin Web)**: React, TypeScript, Vite
